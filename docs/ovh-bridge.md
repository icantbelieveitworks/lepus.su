Debian 9, сеть через `systemd-network`, необходимо сделать bridge.

```
# nano /etc/systemd/network/br0.netdev
[NetDev]
Name=br0
Kind=bridge
MACAddress=0c:c4:7a:d9:c8:b8
```

```
# nano /etc/systemd/network/uplink.network
[Match]
Name=eno1
MACAddress=0c:c4:7a:d9:c8:b8
 
[Network]
Bridge=br0
```
